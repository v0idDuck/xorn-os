# Xorn OS
Xorn OS is a simple operating system written in **C** using **GNU-EFI**.

## 🚀 Features
* **Simple Shell**
  > **IMPORTANT:** Currently, `shell/xsh` is being rewritten to `xeapi` so that it runs as a standalone application.
* **FAT32 Support** — Full implementation developed with the assistance of **Claude AI**.
* **Apps Support** — Basic infrastructure for running external applications.

## 🛠 Development & Build
* **Compiler:** **Clang** (used for building the core and applications).
* **Build System:** Entirely automated using **PowerShell**.
* **AI Assistance:** **Claude (Anthropic)** helped with FAT32 implementation, debugging, and C code optimization.
